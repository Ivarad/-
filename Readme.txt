Для использования программы можно воспользоваться «Установочный файл.exe» и следуя инструкциям установить программу.
Для использования исходных файлов проекта можно воспользоваться Android Studio или Visual Studio Code, установив плагины:
•	Dart;
•	Flutter;
•	Material Icon Theme.
Так же необходимо установить Flutter SDK, на диск, где есть проект, ссылка на скачивание: 
https://docs.flutter.dev/get-started/install
После скачивания и распаковки, необходимо запустить файл «flutter_console.bat», далее надо перейти в папку с проектом в консоли, для этого необходимо воспользоваться командой «cd путь к проекту». После перехода в нужную директорию необходимо прописать команду «flutter clean» для очистки кэша, далее необходимо прописать команду «flutter pub get», далее можно запустить сам проект.
